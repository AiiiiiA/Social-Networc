deploy после изменения проекта для формирования папки чтобы разместить на GitHub:
npm run deploy - создает папку build

git:
git clone ссылкаsshнапроект - крлонирование проекта
git pull - загрузка данных
git checkout названиеветки - выбор ветки

-= изменение файлов =-
git add . - выбор файлов для коммита (вместо точки можо указать название конкретного файла)
git commit -m "описание коммита" - коммит файлов
git push - пуш коммита на github
на github сливаем ветки
